# hxd's debug list

- 线段树要开4倍内存

- cin跑得超级慢

- memset 比 for循环快得多

- 有多组数据时要清空全局变量（以及各种队列）

- `std::accumulate` 的返回类型由第三个参数（初始值）决定。

- 进bfs的时候要记得push初始状态。 / front后记得pop。

- 检查输出编号是否在范围内(0)。

- 检查不存在可行解时的输出(-1)。

- cin不仅跑得慢，而且跑得超级无敌螺旋飞天香蕉船慢。
